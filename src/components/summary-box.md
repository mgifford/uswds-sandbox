# Summary Box

Summary boxes highlight key information with a clear visual hierarchy.

## Default Summary Box

<div class="usa-summary-box" role="region" aria-labelledby="summary-box-heading">
  <h3 class="usa-summary-box__heading" id="summary-box-heading">Key information</h3>
  <div class="usa-summary-box__body">
    <p>This summary box provides important information at a glance. It is useful for highlighting critical details that users need to know.</p>
    <ul class="usa-list">
      <li>First key point about the topic</li>
      <li>Second important detail</li>
      <li>Third relevant item</li>
    </ul>
  </div>
</div>
