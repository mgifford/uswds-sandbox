# Combo Box

A combo box is a combination of a text input and a dropdown list that allows users to select from a list of options or type their own value.

## Default Combo Box

<div class="usa-combo-box">
  <select class="usa-select" name="combo-box-default" id="combo-box-default">
    <option value>Select an option</option>
    <option value="green">Green</option>
    <option value="blue">Blue</option>
    <option value="red">Red</option>
    <option value="yellow">Yellow</option>
    <option value="purple">Purple</option>
  </select>
</div>
