## Slider with Variants

<p>Slider variants include options for different display styles:</p>

<ul class="usa-list">
  <li><strong>Default slider</strong> — Standard carousel with navigation controls</li>
  <li><strong>Slider with thumbnails</strong> — Shows small preview images below</li>
  <li><strong>Slider with dots</strong> — Dot indicators instead of arrows</li>
</ul>

## Slider Accessibility

<p>When using slider components, ensure:</p>
<ul class="usa-list">
  <li>All images have descriptive alt text</li>
  <li>Captions provide context for visual content</li>
  <li>Keyboard navigation is supported</li>
  <li>Pause/autoplay controls are available if auto-rotating</li>
</ul>
