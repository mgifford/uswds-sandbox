# Button Group

Button groups place two or more buttons together as a single unit.

## Default Button Group

<div class="usa-button-group">
  <button class="usa-button usa-button--outline">Back</button>
  <button class="usa-button">Next</button>
</div>

## Button Group with Icon

<div class="usa-button-group">
  <button class="usa-button usa-button--outline">
    ← Previous
  </button>
  <button class="usa-button">
    Next →
  </button>
</div>

## Segmented Button Group

<div class="usa-button-group usa-button-group--segmented">
  <button class="usa-button usa-button--outline">Option 1</button>
  <button class="usa-button usa-button--outline">Option 2</button>
  <button class="usa-button usa-button--outline">Option 3</button>
</div>

## Stacked Button Group

<div class="usa-button-group usa-button-group--stacked">
  <button class="usa-button">Save and Continue</button>
  <button class="usa-button usa-button--outline">Save as Draft</button>
  <button class="usa-button usa-button--unstyled">Cancel</button>
</div>
