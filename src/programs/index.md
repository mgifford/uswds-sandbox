    <div class="grid-row grid-gap">
      <div class="grid-col-8">
        <h3 class="font-heading-lg">Consistency Across Services</h3>
        <p>When citizens interact with government services, they shouldn't have to relearn how to navigate each page. USWDS provides consistent patterns that make government digital services feel familiar and trustworthy.</p>
        
        <h3 class="font-heading-lg">Accessibility Built In</h3>
        <p>Every USWDS component is tested against WCAG 2.1 AA standards. By using the design system, you get accessibility for free—proper color contrast, keyboard navigation, screen reader support, and more.</p>
        
        <h3 class="font-heading-lg">Rapid Development</h3>
        <p>These demos combine multiple USWDS components into realistic pages. Instead of building from scratch, teams can mix and match components to create effective pages in hours, not weeks.</p>
        
        <h3 class="font-heading-lg">Theme Flexibility</h3>
        <p>USWDS tokens allow agencies to customize colors, fonts, and spacing while maintaining the core design principles. This means each agency can express their brand while staying accessible and consistent.</p>
